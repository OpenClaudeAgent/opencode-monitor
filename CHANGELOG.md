# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-26

### Added
- Initial release of OpenCode SwiftBar Monitor
- Event daemon (opencode-eventd) for real-time monitoring
- Usage daemon (opencode-usaged) for Claude API statistics
- SwiftBar plugin for menu bar display
- Automated installation script
- Automated uninstallation script
- Comprehensive documentation
- Example data files
- LaunchAgent configuration for auto-start

### Features
- Real-time OpenCode instance monitoring
- SSE stream listening + HTTP polling fallback
- Claude API usage tracking
- Menu bar display with 2-second refresh
- Click-to-focus iTerm2 integration
- Permission request detection
- Todo task tracking
- Tool execution monitoring
