# OpenCode Monitor - Documentation

Documentation technique du projet OpenCode Monitor.

## Structure

```
docs/
├── design/           # Design system (couleurs, composants)
├── sprints/          # Historique des sprints et user stories
└── README.md         # Ce fichier
```

## Liens rapides

- [Design System](./design/README.md) - Couleurs, typographie, composants
- [Sprints](./sprints/README.md) - Historique des sprints

## Conventions

- **Un dossier par sprint majeur** dans `sprints/`
- **User stories** au format standard (En tant que... Je veux... Afin de...)
- **Fichiers < 500 lignes** pour rester maintenables
